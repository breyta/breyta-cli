# Contributing

This repository is public so Breyta users can inspect, audit, and build the CLI
themselves.

We are not accepting pull requests or code contributions to `breyta-cli` at this
time.

If you need help, want to report a bug, or have product feedback:

- Email `hello@breyta.ai`
- If you already use the CLI, you can also send feedback with `breyta feedback send`

Please do not report security issues in public. See `SECURITY.md`.
